    • Purpose

    The purpose of this project was to analysis how weather patterns may change and alter as you get closer to the equator. To accomplish this analysis, we pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

    After assembling the data, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visulizations created as paer of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
